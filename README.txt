JAVA Version 8 used in this project.

Compiling from Eclipse:
Use main class DataFilePopulator to start the process to create a new file or populate an existing table.
Use main class TableAnalyzer to start the analysis of the file previously created.

Compiling from terminal: macOS
From the project directory, execute command "java -classpath bin p2MainClasses.DataFilePopulator raf"
From the project directory, execute command "java -classpath bin p2MainClasses.TableAnalyzer raf"
